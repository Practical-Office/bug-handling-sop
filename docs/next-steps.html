2 · Homework F

1–2 real bugs end-to-end

Within the next sprint (or agreed window), complete 1–2 production or sandbox bugs using the full process:

  1. Mandatory report template (grill if intake is vague).
  2. /triage — Severity s* + Priority p* + Triage State; work only at ready-for-*.
  3. /diagnosing-bugs — six phases; feedback loop first (red Phase 1 command).
  4. /tdd fix at an agreed seam; re-run original Phase 1.
  5. PR → /code-review (Standards ‖ Spec) → QA verify → Root Cause + Prevention → close.

Prefer your product backlog for homework — not BookIQ work from this training repo. Keep Exercise Lab for class drills only.
